+# 구매 품목 관리·조회 페이지
+
+PowerShare/MS365 등 파일 공유 위치에 `index.html`과 `items.csv`를 함께 두고 열면, CSV만 수정해도 바로 반영되는 구매 품목 검색/조회 페이지입니다.
+
+## 사용 방법
+1. `items.csv`를 동일 폴더에 둔 상태로 `index.html`을 브라우저에서 엽니다.
+2. 상단 검색창, 카테고리 버튼, 태그 필터, 정렬 드롭다운을 조합해 원하는 품목을 찾습니다.
+3. "열기" 버튼을 누르면 구매 링크가 새 탭에서 열립니다.
+4. CSV를 수정한 뒤 페이지를 새로고침하면 변경 사항이 즉시 적용됩니다. (캐시 방지용 timestamp 쿼리 자동 추가)
+
+## CSV 형식
+```
+name,category,price,url,tags
+```
+- **name**: 품명 (필수)
+- **category**: 아래 목록 중 하나 (필수)
+  - 사무용품, 소모품, 주방/탕비/간식, 가전/가구, IT/장비, 숙소/시설용품, 비품(리필항목), 기타
+- **price**: 숫자만 입력, 쉼표 없음 (필수)
+- **url**: 구매 링크 (필수, 새 탭에서 열림)
+- **tags**: 쉼표 구분 다중 태그, 비워도 됨. 사용 가능 태그: 본사, 부산지부, 휴비재, 청평, 동비재, 회의실, 탕비실, 카페, 숙소, 공용, 정기구매, 긴급구매, 복지, 직원, 게스트
+
+> 쉼표가 포함된 태그 목록은 CSV에서 큰따옴표로 감싸 주세요. 예: `"본사, 회의실, 정기구매"`
+
+## 주요 기능
+- 카테고리 탭 전환 기반 테이블 필터링 (페이지 이동 없이 즉시 반영)
+- 다중 태그 선택 및 텍스트 검색(품명/태그) 결합 필터링
+- 가격/이름 기준 정렬, 자동 가격 천단위 표시
+- 모바일 환경에서 태그 열 자동 감추기
+- CSV를 새로 불러오는 새로고침 버튼, fetch 시 timestamp 쿼리로 캐싱 방지
+
